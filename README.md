
# Base Bot Slash

Base modular para bots Discord com carregamento automático de comandos e eventos, registro via API e tratamento de erros integrado.


## Features

- Carregamento automático de slash commands por pasta
- Suporte a subpastas de comandos e eventos
- Registro automático de comandos na API do Discord ao iniciar
- Tratamento de erros em comandos e eventos
- Estrutura organizada e escalável


## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the source

```bash
  node .
```
